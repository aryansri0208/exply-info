# Images Directory

This directory contains all screenshots and images for the Exply website.

## Directory Structure

```
images/
├── hero/
│   └── exply-hero-demo.png          # Main hero screenshot showing Exply in action
├── how-it-works/
│   ├── step-1-highlight.png         # Screenshot: User highlighting text
│   ├── step-2-click.png             # Screenshot: Right-click menu or extension icon
│   └── step-3-explanation.png       # Screenshot: Inline explanation popup
└── use-cases/
    ├── explain-mode.png             # Screenshot: Explain mode in action
    ├── simplify-mode.png            # Screenshot: Simplify mode in action
    ├── implications-mode.png        # Screenshot: So What? mode in action
    └── translate-mode.png           # Screenshot: Translate mode in action
```

## Image Specifications

### Hero Image
- **Recommended size:** 1200x800px or larger
- **Format:** PNG or WebP
- **Content:** Screenshot of Exply extension showing an inline explanation popup on a real webpage
- **Aspect ratio:** 3:2 or 16:10

### How It Works Images
- **Recommended size:** 800x600px each
- **Format:** PNG or WebP
- **Content:** 
  - Step 1: Text highlighted on a webpage
  - Step 2: Context menu or extension popup showing "Explain with Exply"
  - Step 3: Full explanation popup visible inline on the page
- **Aspect ratio:** 4:3

### Use Cases Images
- **Recommended size:** 1000x700px each
- **Format:** PNG or WebP
- **Content:** Screenshots showing each mode (Explain, Simplify, So What?, Translate) in action
- **Aspect ratio:** 3:2 or 10:7

## Notes
- All images should be optimized for web (compressed but high quality)
- Include alt text in HTML for accessibility
- Consider using WebP format with PNG fallback for better performance
- Images should match the actual Exply extension UI from the screenshots provided
