# Logo Directory

Place your Exply logo image file here.

## File Name
- **exply-logo.png** (or .svg, .jpg, .webp)

## Recommended Specifications
- **Format**: PNG with transparency (preferred) or SVG
- **Size**: At least 64x64px for retina displays
- **Aspect Ratio**: Square or horizontal rectangular logo
- **Background**: Transparent background recommended

## Usage
The logo will be displayed in:
- Navigation bar (desktop: logo + text, mobile: logo only)
- Height: 32px on desktop, 28px on mobile

## Supported Formats
- PNG (recommended with transparency)
- SVG (for scalability)
- WebP (for performance)
- JPG (if no transparency needed)

Simply save your logo file as `exply-logo.png` in this directory and it will automatically appear in the navigation.
