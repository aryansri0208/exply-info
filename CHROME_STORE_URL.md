# Chrome Web Store URL

To connect your website to your Chrome Web Store listing:

1. Get your Chrome Web Store URL:
   - Go to https://chrome.google.com/webstore
   - Search for "Exply" or go to your extension's listing
   - Copy the full URL (it will look like: `https://chrome.google.com/webstore/detail/exply/YOUR_EXTENSION_ID`)

2. Replace the placeholder URL in these files:
   - `index.html` (2 places: hero button and footer)
   - `use-cases.html` (2 places: CTA button and footer)
   - `coming-soon.html` (footer - optional, you may want to remove this page)

3. Search for: `YOUR_CHROME_WEB_STORE_URL_HERE`
   - Replace with your actual Chrome Web Store URL

Example:
- Find: `href="YOUR_CHROME_WEB_STORE_URL_HERE"`
- Replace with: `href="https://chrome.google.com/webstore/detail/exply/abcdefghijklmnopqrstuvwxyz"`
